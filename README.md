# Tic-Tac-Toe Game

A classic Tic-Tac-Toe game implementation in Python with an interactive command-line interface.

## Features

- Two-player gameplay
- Simple and intuitive board display
- Input validation and error handling
- Win/draw detection
- Replay functionality

## Requirements

- Python 3.6 or higher

## Installation

1. Clone the repository:
```bash
git clone https://github.com/AS-Developer-17/Tic-Tac-Toe-Game.git
cd Tic-Tac-Toe-Game
```

2. Run the game:
```bash
python tic_tac_toe.py
```

## How to Play

- The game is played on a 3x3 grid
- Player X goes first, followed by Player O
- Players take turns entering their position (1-9)
- The first player to get three marks in a row (horizontal, vertical, or diagonal) wins
- If all squares are filled with no winner, the game is a draw

## Game Positions

```
 1 | 2 | 3
-----------
 4 | 5 | 6
-----------
 7 | 8 | 9
```
